## 🚀 Quick Start

### Prerequisites
- Node.js 18+ 
- npm, pnpm, yarn, or bun

### Installation

Install dependencies:

```bash
npm install
```

Or using other package managers:

```bash
pnpm install    # pnpm
yarn install    # yarn
bun install     # bun
```

### Development Server

Start the development server on `http://localhost:3000`:

```bash
npm run dev
```

Or with other package managers:

```bash
pnpm dev        # pnpm
yarn dev        # yarn
bun run dev     # bun
```

### Building for Production

Build the application for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

Or with other package managers:

```bash
pnpm build      # pnpm
yarn build      # yarn
bun run build   # bun
```

Generate static deployment:

```bash
npm run generate
```
